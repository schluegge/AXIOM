# Legacy AXIOM boundary

The repository state at commit `67bc156a6afdb12654fe8cbb6837e5d449cfd063`
is the frozen AXIOM v0.7 legacy baseline for AXIOM Next.

This boundary preserves normative semantics, contract tests, and benchmark
contracts as evidence and requirements. It does not promote the Python/LLVM
compiler, custom runtime, review automation, or workflows into the AXIOM Next
production architecture.

`legacy/provenance.json` is generated by `tools/legacy_inventory.py`. Only files
listed by that tool are approved for semantic reuse. Importing implementation
code from a reset candidate requires a separate reviewed decision and new tests.
