# AXIOM Next MVP release gate

Status: normative

The MVP is not a language release. It may be promoted only after the golden
application passes source, HIR, interpreter, component, capability, deterministic
artifact, Windows, Linux, benchmark replay, and evidence-bundle gates.

The final machine decision is exactly one of:

- `PROMOTE_AXIOM_NEXT`
- `RETAIN_AS_EXPERIMENT`
- `REJECT_ARCHITECTURE`

Tasks 0–2 satisfy only the project-boundary prerequisite and cannot produce a
promotion decision.
